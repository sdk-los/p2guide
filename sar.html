Source Auto Record

Source Auto Record(SAR) - это плагин, который разработал NeKz. Он добавляет множество полезных комманд и улучшений.

Установка

  1. Последнюю версию можно скачать здесь. Нужен файл sar.dll
  2. Поместить файл в папку steam/steamapps/common/Portal 2/portal2
  3. Перезапустите игру если она была запущена
  4. После загрузки игры введите в консоль - plugin_load sar

Команды

Уменьшение времени загрузки уровней

Эти команды отключают анимацию и визуальные эффекты во время загрузки уровней, что ускоряет загрузки.

  • ui_loadingscreen_fadein_time 0; ui_loadingscreen_mintransition_time 0; ui_loadingscreen_transition_time 0

Отключение всплывающего окна в режиме испытаний

Эта команда автоматически закрывает всплывающее окно в режиме испытаний.

  • sar_disable_challenge_stats_hud 1

Демо-заись

Эта команда автоматически начинает запись demo-файлов на каждом уровне.

  • sar_autorecord 1

Смещение времени для Container Ride Save

Эта команда смещает время таймера, для учета пропущенной кат-сцены в самом начале игры.

  • sar_speedrun_offset 16868

Сброс времени

Эта команда сбрасывает время таймера

  • sar_speedrun_reset

Учет времени во время паузы

Эта команда позволяет не учитывать время во время паузы, если значение равно 0, и наоборот.

  • sar_speedrun_time_pauses 0

Предотвращение немедленного запуска таймера

Эта команда предотвращает немедленный запуск таймера при загрузке на уровень.

  • sar_speedrun_autostart 0
